# Occurity 3D prints
These 3D models are provided for convenience. If used with a Flirc Raspberry Pi case they enable you to mount the Raspberry Pi on the wall above a monitor for a clean and easily accessible solution.

## Raspberry Pi holder
This is a Raspberry Pi wall mount holder. The Raspberry Pi slides easily into the holder allowing easy access to ports and the SD card slot in case of maintenance. The print is designed for the Flirc Raspberry Pi case.

## Raspberry Pi spacer bracket / template
This print serves as a distance template helping to provide a standardized placement of the Raspberry Pi holder above a monitor. Temporarily place the template bracket at the top-left corner of the monitor. Place the Raspberry Pi holder into the bracket. Screw the holder into the wall and remove the template again.
